# Date

date 命令所支持的格式选项

|   日期内容   |  格式  |
|:-----------:|:------:|
|    工作日    | %a或%A |
|      月      | %b或%B |
|      日      |   %d   |
| 特定格式日期 |   %D   |
|      年      | %y或%Y |
|     小时     | %I或%H |
|     分钟     |   %M   |
|      秒      |   %S   |
|     納秒     |   %N   |
|  Unix纪元时  |   %s   |
